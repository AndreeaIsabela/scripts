# Scripts

A collection of reusable Python utility scripts.

## Prerequisites

- **Python 3.10+** — [Download here](https://www.python.org/downloads/)

## Scripts

| Script | Description |
|---|---|
| [`image_convertor/`](./image_convertor/) | Convert images between formats (PNG, JPEG, WebP, etc.) |
| [`email_sender/`](./email_sender/) | Send HTML emails from the CLI using customisable templates |
| [`password_checker/`](./password_checker/) | Check passwords against known data breaches via the HIBP API |
| [`message_sender/`](./message_sender/) | Send SMS or WhatsApp messages from the CLI via Twilio |
| [`check_website_stock/`](./check_website_stock/) | Log into a shop, check product availability, and get notified via SMS/WhatsApp when items are in stock |

## Getting Started

Clone the repository and navigate into the folder of the script you want to use. Each script folder contains its own README with setup and usage instructions.

```bash
git clone https://github.com/isabelaandreeabala/scripts.git
cd scripts
```
